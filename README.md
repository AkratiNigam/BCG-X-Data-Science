# BCG-X-Data-Science
The project focuses on solving a business problem related to customer churn prediction for a utility company. The objective is to analyze customer data, engineer meaningful features, build predictive models, and generate insights that can help the business proactively reduce churn.

Through this simulation, I practiced the end-to-end data science workflow including data exploration, feature engineering, model building, and business insight generation.

## Problem Statement

Customer churn is a critical challenge for energy providers. Losing customers leads to revenue loss and increased acquisition costs.

-The goal of this project is to:

-Analyze customer and pricing data

-Identify factors influencing customer churn

-Build a machine learning model to predict customers likely to churn

-Provide business recommendations based on data insights

-The final deliverable simulates a consulting-style solution where technical analysis is translated into actionable business insights.

##Project Tasks

The simulation consists of the following key tasks:

1. Exploratory Data Analysis (EDA)

Loaded and inspected the datasets

Cleaned and preprocessed missing or inconsistent data

Explored customer behavior patterns

Analyzed distributions, correlations, and feature importance

2. Feature Engineering

Created new features from raw data

Calculated price sensitivity indicators

Engineered features related to customer tenure, usage patterns, and pricing changes

3. Predictive Modeling

Built a classification model to predict churn

Trained and validated the model using historical data

Evaluated model performance using standard metrics

4. Business Insights

Identified key drivers of churn

Translated model outputs into strategic recommendations

Presented findings in a clear and concise manner
